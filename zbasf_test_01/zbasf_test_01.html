REPORT ZBASF_TEST_01.

DATA(sentence) = `AB비P is excellent `.

cl_demo_output=>write( |Number of words: { count( val = sentence regex = `[\\S|비|A-Z|a-z|\\']+` ) }| ).

DO count( val = sentence regex = `[\\S|비|A-Z|a-z|\\']+` ) TIMES.

  DATA(lv_str) = match( val = sentence regex = `[\\S|비|A-Z|a-z|\\']+` occ = sy-index ).

  DATA(lv_unique_str) = REDUCE string( INIT s = lv_str
                                                           FOR i = 0 UNTIL i >= strlen( lv_str )
                                                           LET x = lv_str+i(1) IN
                                                         NEXT s = condense( val = replace( val = s sub = x with = space occ = 2 ) del = space ) ).

  cl_demo_output=>write( |Number of unique characters in the word: { match( val = sentence regex = `[\\S|비|A-Z|a-z|\\']+` occ = sy-index ) }| && | - | && |{ strlen( lv_unique_str ) }| ).

ENDDO.

cl_demo_output=>display( ).

*why AB비P correctly has 4 unique letters.
*A is HalfWidthAlphanumeric, 비 is FullWidthAlphanumeric, so are two different letters.
